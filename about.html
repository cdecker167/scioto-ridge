Our Mission

This site was created for the Bussiness Professionals of America 2021 Web Application competition.

Scioto Ridge is a newly acquired theme park by High Thrills Entertainment LLC. Our web development team, consisting of members of the Pickerington High School North chapter of BPA Garrett Cox, Colin Decker, Logan Nowlin, and Tyler Brunelle.

Garrett Cox developed the backend of the site by creating a Rest API with the Express module of Node.js. The API handles different requests from the frontend of the website, and accesses the database accordingly.

Colin Decker hand-coded the frontend of the website using complex html, css, and JavaScript for both the users and administrators

Logan Nowlin created the assets used in the website using primarily Adobe Illustrator. Logan created the graphics for each of the attractions, as well as the map of the park and other assets across the site.

Tyler Brunelle was in charge of the user interface. He had to combine functionality and style. Tyler started by choosing a color scheme, and then created mockups for the sites pages. He materialized new and unique ways to style and display the reservation system and park information.
